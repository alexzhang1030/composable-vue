# Composition vs Options

Vue3 和 Vue2 有什么本质上的区别？

大部分人肯定会想到，Vue3 引入了 Composition API，这是一个破坏性改动，这标志着 Vue3 不再向后兼容 Vue2。但最重要的是整个心智模型已经发生了变化。

探究其源，Vue2 是如何运行起来的？

```vue
<script>
export default {
  data() {
    return {
      vue2: 'vue2'
    }
  },
  methods: {
    foo() {
      return 'vue2'
    }
  }
}
</script>

<template>
  {{ vue2 }}
</template>
```
